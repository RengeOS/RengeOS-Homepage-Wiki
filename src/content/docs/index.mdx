## Features

<CardGrid>
	<Card title="Minimal yet powerful" icon="rocket">
	An opinionated, minimal Arch-based system focused on performance, featuring a custom XanMod kernel and LZ4 compression.
	</Card>
	<Card title="Rollback any time you want" icon="star">
		Bcachefs filesystem is supported by default, which is well-known for its CoW and Snapshots like Btrfs filesystem but more effective
	</Card>
	<Card title="Install once" icon="heart">
		There's no need to reinstall if things get messed up with `ros-reborn` in `recovery mode` on GRUB
	</Card>
	<Card title="Your system, your rules" icon="setting">
		You decide what packages gets installed and enabled so have fun with your minimal Arch-based system
	</Card>
</CardGrid>
